# GeoLite2toGeoLiteLegacy

This script generates GeoLiteLegacy database (.dat) from GeoLite2 CSV files. 
  * https://dev.maxmind.com/geoip/geoip2/geolite2/
  * https://dev.maxmind.com/geoip/legacy/geolite/

## Requirements
  * Python 2.7

Setup
------------------------------------------------------
    $ ./setup.sh

Usage
------------------------------------------------------
    $ ./generate.sh
